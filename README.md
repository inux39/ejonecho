## Usage
```
$ ejonecho "グエー"
:ejoneco: < グエー
```

```
$ ejonecho "ウヒョー" -t
:ejoneco: < ウヒョー4時
```

```
$ echo "ドヒャー" | ejonecho -t
:ejoneco: < ドヒャー4時
```

## Build
